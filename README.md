# PF2
Repository of the scripts used for the Phase Folding and Period Finding (PF^2) algorithm, used in Tramuto et al. (submitted to ApJ in June 2026).
This repository contains the zip file of the PFˆ2 algoirthm (DOI: https://doi.org/10.20371/inaf/SW/2026_00014), as well as the interactive plots (DOI: https://doi.org/10.20371/inaf/SW/2026_00015) extracted from the ones present in Tramuto et al. (submitted to ApJ in June 2026).
